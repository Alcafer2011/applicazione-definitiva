NETWORK TOPOLOGY
================

All services communicate via "bos-net" bridge network (Docker internal DNS).

    Frontend Apps ──► Traefik (DNS: bos-traefik) ──► Services (all use bos-net)
                           │
                           ├──► MongoDB (bos-mongo)
                           ├──► RabbitMQ (bos-rabbitmq)
                           ├──► Loki (bos-loki)
                           ├──► Promtail (bos-promtail)
                           └──► Grafana (bos-grafana)


SERVICE DEPENDENCIES
====================

Critical Path:
  ApiGateway → Identity.Service → Audit.Service
            → Governance.Service
            → Message.Service → RabbitMQ

AI Pipeline:
  Orchestrator → DigitalBrain.Service (port 5001)
  DigitalBrain → MultiAI.Service
              → OcrDdt.Service

Data Pipeline:
  AnalyticsHub → SearchEngine.Service
              → BI.Service
              → MemoryGraph.Service
