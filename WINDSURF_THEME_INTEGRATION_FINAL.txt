🚀 QUICK START

For React:
  1. Wrap App with ThemeProvider:
     <ThemeProvider><App /></ThemeProvider>

  2. Use hook in components:
     const { isDarkMode, toggleTheme, getColor } = useTheme()

  3. Apply colors:
     background: getColor('bgGlassHeavy')
     border: `1px solid ${getColor('borderCrystal')}`

For HTML/JavaScript:
  1. Link theme.css:
     <link rel="stylesheet" href="/css/theme.css">

  2. Load theme manager:
     <script src="/js/theme-manager.js"></script>

  3. Use API:
     themeManager.toggleTheme()
     themeManager.getColor('neonCyan')
     themeManager.setTheme('light')

🔧 THEME CUSTOMIZATION

To Create a Custom Theme:

```javascript
// Create new theme
const myTheme = themeManager.createCustomTheme('custom', {
  ...themeManager.themes.dark.colors,
  neonCyan: '#00ff00',      // Override cyan
  bgVoid: '#1a1a2e',        // Override background
  customColor: '#ff0000'    // Add new color
});

// Activate custom theme
themeManager.setTheme('custom');
```

To Override CSS Variables:

```javascript
// Direct CSS override
document.documentElement.style.setProperty('--neon-cyan', '#00ff00');
document.documentElement.style.setProperty('--text-primary', '#f0f0f0');
```
