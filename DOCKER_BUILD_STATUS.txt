TROUBLESHOOTING IF BUILD FAILS
═════════════════════════════════════════════════════════════════════════════════

If NuGet connectivity doesn't recover:

1. Check network connectivity:
   - ping api.nuget.org
   - Test proxy settings if behind firewall

2. Check Docker daemon:
   - docker system info
   - docker ps

3. Clear cache and retry:
   - docker-compose down -v
   - .\docker-rebuild.ps1 -Action full

4. Build individual service:
   - cd ./AdminConsole.Service
   - docker build -t bos-adminconsole:latest .

5. Check NuGet status:
   - Visit: https://status.nuget.org/
